# AI-Powered Fraud Detection System (Java)
A 99.98% accurate fraud detection model built using the J48 Decision Tree algorithm.

## Features
- Real-time classification of credit card transactions.
- Visual Decision Tree flowchart using Java Swing.
- Automated data preprocessing via Weka.

## How to Run

https://drive.google.com/file/d/1OLfYEXCZ8_cr9-Wo51A0tJ2POWBiJXFy/view?usp=sharing
Note: Download the dataset from this link and place it in the project folder before running.

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Wait for Maven to import dependencies (Weka).
4. Run `FraudDetector.java`.
